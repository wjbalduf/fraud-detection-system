
The dataset is not included because it exceeds GitHub's file size limits.

data/paysim.csv

Run the notebooks in numerical order:
01_EDA.ipynb
02_Feature_Engineering.ipynb
03_Modeling.ipynb
04_Model_Evaluation.ipynb
05_Business_Insights.ipynb

Running 03_Feature_Engineering.ipynb will generate paysim_engineered.csv automatically.
